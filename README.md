# 🐍 Python Quiz App

## 📌 Project Overview
The Python Quiz App is an interactive Online Quiz System developed using Python and Tkinter. This application allows users to answer multiple-choice questions related to Python programming in a simple and user-friendly environment.

The system provides instant feedback for every answer along with explanations, helping users understand concepts and improve their learning experience.


## ✨ Features
✅ Multiple-choice quiz system  
✅ Simple and user-friendly GUI  
✅ Instant correct/wrong feedback  
✅ Explanation for every answer  
✅ Final score calculation  
✅ Review incorrect answers  
✅ Question progress tracking  
✅ Performance-based result message  


## 🛠 Technologies Used
🐍 Python  
🖼 Tkinter GUI Library  
💻 Visual Studio Code  
🌐 GitHub  


## 🎯 Project Objective
The main objective of this project is to create an engaging quiz platform that not only evaluates users but also helps them learn through explanations and mistake review features.
